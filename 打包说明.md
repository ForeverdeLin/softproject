# 打包成EXE文件说明

## 第一步：安装PyInstaller

```powershell
pip install pyinstaller
```

## 第二步：测试打包命令

先测试一下打包命令是否能正常工作：

```powershell
pyinstaller --onefile --name "失物招领系统" build_exe.py
```

**参数说明：**
- `--onefile`: 打包成单个exe文件
- `--name`: 指定exe文件名

## 第三步：检查打包结果

打包完成后，会在 `dist` 文件夹中生成exe文件。

**先测试这一步，看看有没有错误。如果有错误，告诉我具体的错误信息。**
