# 校园失物招领系统

## 项目简介

基于规则型智能体的校园失物招领系统，实现失物与招领信息的智能匹配。

## 快速开始

```powershell
# 安装依赖
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt

# 启动服务器
python -m app.main

# 访问
# 浏览器打开 http://localhost:5000
```

## 核心功能

- ✅ 用户注册/登录/登出
- ✅ 发布失物/招领信息
- ✅ 智能体自动匹配（5条匹配规则）
- ✅ 查看匹配结果（按匹配度排序）
- ✅ 通知系统（匹配通知、提醒通知、系统公告）
- ✅ 完整的Web界面（12个页面）
- ✅ RESTful API接口（支持客户端共享数据）
